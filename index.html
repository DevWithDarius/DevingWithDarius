Project Overview

This web application is deployed on Amazon Web Services (AWS) using a combination of EC2, an Application Load Balancer (ALB), VPC, and IAM policies to ensure a secure and scalable infrastructure.

Key Features

  • EC2 Instance with secure HTTPS configuration
  • SSL/TLS Encryption via AWS ACM (Certificate Manager)
  • CloudWatch Monitoring for Logs and Alerts
  • Web Application Firewall (WAF) for Input Validation and Security
  • Custom Domain: devwithdarius.us

AWS Architecture

This project follows AWS best practices for deploying secure web applications, and ensures proper security configurations, such as:

  • IAM Least Privilege Access
  • Secure VPC Setup with Public and Private Subnets
  • Automatic Load Balancing with ALB
